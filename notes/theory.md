- **Y結線**：線間電圧 = √3 × 相電圧、線電流 = 相電流

#### 半導体・ダイオード

**pn接合の基本**
- p型（正孔多数）とn型（電子多数）を接合すると境界に**空乏層**ができる
- 順バイアス（p側を＋）→ 空乏層縮小 → 電流が流れる
- 逆バイアス（p側を−）→ 空乏層拡大 → 電流ほぼ流れない（一方通行の弁）

### なぜ (1) が正解か ── 3つの空欄を1つずつ潰す

#### （ア）ゲート ← なぜドレインやソースではないか

FETの3端子の役割：
```
ゲート  (G) ── 制御する端子（電圧を加える）
ドレイン(D) ── 電流が入ってくる端子
ソース  (S) ── 電流が出ていく端子
```
「電界によって制御」するのは**ゲート電圧**。ドレイン・ソースは電流の出入り口であって制御用ではない。

**記憶の軸**：「G = Gate = 門番。門番が電圧で扉の開け閉めをする」

---

#### （イ）MOS ← FETの2種類を覚える

| 種類 | 正式名称 | ゲート構造 | 特徴 |
|------|---------|-----------|------|
| **接合形（JFET）** | Junction FET | ゲートとチャネルがpn接合 | 逆バイアスで空乏層を広げてチャネルを絞る |
| **MOS形（MOSFET）** | Metal Oxide Semiconductor FET | ゲートと半導体の間に酸化膜（絶縁体）を挟む | 電界でチャネルを誘起する |

誤答の「MSI」「DIP」は半導体パッケージ・ICの用語であって、FETの種類ではない。

**記憶の軸**：「FETは接合形とMOS形の2種類だけ」

---

#### （ウ）nチャネル ← 図記号の読み方

JFETの図記号でnチャネルかpチャネルかを判定する方法：

```
nチャネルJFET          pチャネルJFET
    D                     D
    │                     │
G ──┤ →                G ──┤ ←
    │                     │
    S                     S

矢印がチャネルへ向かう → nチャネル
矢印がチャネルから外を向く → pチャネル
```

**記憶の軸**：「矢印が内向き（チャネルへ刺さる）= n型」

---

### FETの動作原理（なぜゲート電圧で電流が変わるか）

#### JFETの場合

```
VGS = 0V → チャネル最大幅 → 最大電流 IDSS
VGS を負にする → ゲートpn接合が逆バイアス → 空乏層が広がる → チャネルが狭くなる
VGS = VP（ピンチオフ電圧、負の値）→ 空乏層がチャネルを完全に塞ぐ → ID ≈ 0
```

**ドレイン電流式（飽和領域）**

```
ID = IDSS × (1 - VGS/VP)²
```

符号の計算例：IDSS = 8mA、VP = −4V、VGS = −2V
```
1 - VGS/VP = 1 - (−2)/(−4) = 1 − 0.5 = 0.5   ← (−)÷(−) = (+) に注意
ID = 8 × 0.5² = 8 × 0.25 = 2mA
```

---

#### MOSFETの場合（エンハンスメント型・試験頻出）

```
VGS < Vth（閾値電圧）→ チャネルなし → ID = 0（OFF）
VGS > Vth           → 電界でチャネルが誘起される → ID 流れる（ON）
```

**ドレイン電流式（飽和領域）**

```
ID = k × (VGS - Vth)²
```

JFETとの違い：ゲートと半導体の間に酸化膜があるため、ゲートに電流は**全く流れない**

---

### FET全種類まとめ

| 分類 | 種類 | ノーマリ | 制御方法 | 式 |
|------|------|---------|---------|---|
| JFET | nチャネル | ON | VGS を負にして絞る | ID = IDSS(1−VGS/VP)² |
| MOSFET | エンハンスメント型（n-ch） | OFF | VGS > Vth で起動 | ID = k(VGS−Vth)² |
| MOSFET | デプレッション型（n-ch） | ON | JFETと同様に動作可 | （同上） |

---

### 試験対策：穴埋め問題の確実な取り方

電験3種の半導体問題は**用語・図記号・分類**を問う穴埋めが中心（計算より頻出）。

**必ず答えられるようにする3点**

1. FETを制御するのは**ゲート電圧**（ドレイン・ソースではない）
2. FETの2種類は**接合形とMOS形**（他の選択肢は半導体パッケージの用語）
3. 図記号の矢印が**内向き = nチャネル**、外向き = pチャネル
